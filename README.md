## 轨道智能探伤小车｜比赛展示网页（静态多页）

这是一个用于**商业挑战赛/路演展示**的静态网站模板，用来介绍“轨道智能探伤小车”项目：产品亮点、技术方案、应用场景、商业模式、团队等。

### 目录结构

- `index.html`：比赛用首页（一句话价值主张 + 核心亮点 + CTA）
- `product.html`：产品介绍（功能/规格/流程）
- `tech.html`：技术方案（传感器、算法、边云协同、可靠性）
- `business.html`：商业模式与落地（客户、价值、成本、竞争）
- `team.html`：团队与合作/联系方式
- `assets/`：样式、脚本、图片（含 SVG 占位）

### 本地打开方式（无需安装任何环境）

1. 进入目录 `rail-inspection-demo`
2. 直接双击打开 `index.html`

> 建议使用 Chrome/Edge 打开。

### 你需要替换/补充的内容（最重要）

- 项目名称、口号、学校/实验室名称
- 关键指标（检测速度、最小缺陷尺寸、定位精度、续航、误报/漏报）
- 实拍照片/视频链接（可放到 `assets/img/` 并替换页面中的占位图）
- 团队成员、指导老师、合作单位、联系方式
